# ICRA 2019 Simulator

## Requirements

- Python3
- OpenAI gym
- Box2D
- swig

## Installation

```
sudo apt-get install swig # or install from source
pip3 install gym box2d box2d-kengz
```

## Run

```
python3 main.py
```
